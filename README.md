# Содержание

1. Checkbox по умолчанию true and false.
2. Вывод состояния Checkbox из state через тернарный оператор.
3. По клику на кнопку, если чекбокс отмечен, выводится button is checked, а если не отмечен - button is NOT checked.
4. С помощью трех чекбоксов выбираем языки, которые знаем: html, css и js. Результат выбора по каждому языку выводим в отдельный абзац.
5. Checkbox и условный рендеринг, сделаем так, чтобы в зависимости от отметки чекбокса, на экран выводился либо один кусочек кода, либо другой. Используем для этого условный рендеринг.
6. С помощью чекбокса спрашиваем у пользователя, если ли ему уже 18 лет. Если чекбокс отмечен, то: Ура есть 18, если нет: увы вам еще нет 18.
7. Дан чекбокс и абзац. Если чекбокс отмечен, пусть абзац будет видимым на экране, а если не отмечен - прячем его.
